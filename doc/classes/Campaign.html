wpc_setup(actions = 'END', wpc_id_or_names = nil, is_ok_missing_wpc = nil)

Web Publisher Campaign Setup. Returns names (or id) of the acted WPCs. 

 action(s):: (Array of ) action string(s), applied sequentially when the
       action is applicable.  Default: END
 wpc_id_or_names:: An integer WPC_ID or (array of) WPC name string(s)
       All names which RegExp match to the input string will be acted on.
      If omitted, setup all existing WPCs.
      If first name is '-', setup all WPCs except for listed ones.
 is_check_wpc_names:: If true, raises error unless input WPCs exist.
      Otherwise ignores WPCs that are not found.

Example: 

 campaign.wpc_setup("PROVISION") # Provision all WPCS that can be provisioned
 campaign.wpc_setup %w[END STOP], "Yahoo Site Submit DMA" # END or STOP Yahoo
 campaign.wpc_setup "PROVISION,AUTO RUN", "-,Google" # Provision and AutoRun
